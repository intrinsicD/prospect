## Goal

Obtain the first valid real answer to whether a smooth affine deformation, global
channel appearance change, or their combination can make a genuinely causal
half-second prediction on the existing eight Perception Test videos. Fit only
`previous -> current`, freeze all predictions, and expose `future` only to the scorer.

This goal was intentionally not executed on real frames: the preregistered generated
off-grid gate rejected the finite-grid translation candidate first.

## Non-goals

- No target-aware fitting, target-adaptive research loop, population claim,
  end-to-end Prospect capability claim, multimodal fusion, planning, or control.
- No repair, retry, reuse, deletion, or reinterpretation of the immutable MM-009
  canonical root. MM-011 is a new identity and its first possible scientific run.
- No scientific algorithm, threshold, baseline, decision-order, panel, or fold change
  from the pre-real-validated MM-009 assay.
- No historical MM-001--MM-007 verifier import or execution. LCV-001 replaces that
  lineage/runtime surface.
- No AIDE2 adaptation on the eight real futures. A harness comparison needs several
  repeatable development tasks and a physically withheld one-shot judge.
- No edit to `src/prospect/`, the sealed MM-001--MM-009 result packages, or the pinned
  MM-008 v2.2 mathematical core.

## Interface to satisfy

Experiment-owned typed functions under `bench/multimodal_causal_assay/`:

The package is a task-specific benchmark harness. It adds no production protocol.

## Approach

Mechanically rebind the already audited MM-009 scientific engine to MM-011 schema,
hash, context, classification, and result identities. Replace its two failed
infrastructure assumptions:

1. copy and validate the complete canonical 42-file/17-directory LCV-001 package,
   then consume the MM-007 frame archive and parent-closure contract from that copy;
2. use the LCV lexical-execution supervisor pattern so the terminal CPython binary
   runs with literal `.venv/bin/python` as `argv[0]`, then revalidate NumPy 2.4.6,
   one-thread OpenBLAS, source maps, and the SVD canary inside the actual child.

Keep the exact 477-frame/453-row panel, four source-current normalizers, three
operator families, prediction/target detachment, persistence/velocity/current-bias
baselines, checkerboard/shuffle/reverse/derangement controls, thresholds, and
decision precedence. Run a small generated off-grid sensitivity audit before freeze;
it may kill the candidate but may not tune on real targets or silently change the
formal question.

## Gate result

Terminal pre-real NO-GO. The immutable generated receipt
`docs/research/2026-07-16-mm011-offgrid-sensitivity-audit.json` records
`ABANDON_FINITE_GRID_BEFORE_REAL_DATA` with evidence digest
`adaf0c4221aa0f99073c337ee9376366ea64d0189a84e4eb4f6798fcbbc9a3c8`.
All six cells passed activity, historical identification, forward improvement, and
bit-exact replay. A1--A3 also passed directionality. T1--T3 failed only directionality:
their forward error was `0.5741`, `0.6243`, and `0.5619` times persistence, but the
same forecasts also approached the previous frame at `0.5948`, `0.6908`, and `0.7518`
times persistence. Both identity/low-texture controls passed, and a clean-process
semantic regeneration was bit-exact. The independent scientist pass in
`docs/research/2026-07-16-mm011-offgrid-sensitivity-result-audit.md` accepted this
bounded disposition after separate point-loop sampling, arithmetic reconstruction,
file-open tracing, and another clean replay.

Therefore the apparent off-grid translation gain is a nondirectional interpolation
effect, not admissible causal extrapolation. No LCV/MM-007 scientific frame was read,
no canonical MM-011 root was created, and no real prediction or target score exists.
The frozen successor is a new-identity continuous-registration assay, not a relaxed
MM-011 threshold or administrative rerun. Independent lifecycle review also found
that any successor must durably bind formal/semantic cleanup and execute copied,
bytecode-free controller sources before a real run.
