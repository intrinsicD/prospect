# Evaluation Rubric

Score each dimension from 0–4.

## Trigger precision

- 0: activates for routine coding or fails on explicit research ideation
- 2: usually activates correctly
- 4: clear boundaries and reliable activation

## Diversity

- 0: one fashionable lineage repeated
- 2: several methods but same conceptual grammar
- 4: independent lanes, incompatible assumptions, and a Pareto portfolio

## Transformational depth

- 0: only A+B combinations
- 2: some assumption changes without distinct predictions
- 4: changed primitives/grammar with irreducible deltas and predictions

## Cross-domain rigor

- 0: metaphor and buzzwords
- 2: plausible analogy with partial mapping
- 4: explicit structural map, preserved mechanism, mismatches, adaptation, and adoption barrier

## Evidence generation

- 0: prose-only hypotheses
- 2: experiments without strong controls
- 4: new observables, null hypotheses, promotion rules, and reproducible controls

## Novelty calibration

- 0: absolute novelty claims
- 2: caveats but weak search disclosure
- 4: adversarial facet audit, nearest work, cutoff, inaccessible sources, and uncertainty range

## Falsifiability

- 0: no decisive test
- 2: broad experiment plan
- 4: cheapest killing test with abandonment criterion and conventional alternative

## Repository compatibility

- 0: ignores architecture and workflow
- 2: mentions constraints
- 4: grounded in files and hands selected ideas into the repository's normal task/ADR/benchmark process
